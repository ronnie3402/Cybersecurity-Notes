## 5. PASSWORD CRACKING TOOLS

---

## Hydra

Brute-force login attack tool used on online services.

### Targets:

- SSH  
- FTP  
- HTTP forms  

### Use case:

- Testing weak credentials  
- Password guessing attacks  
- Service login auditing  

---

## John the Ripper

Offline password cracking tool.

### Used for:

- Hash cracking (offline attacks)  
- Dictionary attacks  
- Brute-force password recovery  

### Key idea:

Works on extracted password hashes instead of live systems.
