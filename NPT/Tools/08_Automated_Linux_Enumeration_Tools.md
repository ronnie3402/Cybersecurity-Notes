## 8. AUTOMATED LINUX ENUMERATION TOOLS

---

## 8.1 LinPEAS

Most powerful Linux enumeration tool.

### Detects:

- misconfigurations  
- SUID binaries  
- cron jobs  
- privilege escalation vectors  

### Key feature:

- Color-coded output  
- Critical findings highlighted automatically  

---

## 8.2 LinEnum

Lightweight Bash enumeration script.

### Shows:

- users  
- SUID binaries  
- running processes  
- system configuration details  

---

## 8.3 Linux Exploit Suggester (LES)

Kernel-based exploit matching tool.

### Function:

- matches kernel version with known CVEs  
- suggests possible exploits  

---

## 8.4 LSE (Linux Smart Enumeration)

Structured enumeration tool.

### Features:

- verbosity levels (0–2)  
- categorized output  
- highlights critical findings automatically  

---

## 8.5 Linux Priv Checker

Python-based enumeration tool.

### Checks:

- weak file permissions  
- cron jobs  
- SUID binaries  
- system misconfigurations
