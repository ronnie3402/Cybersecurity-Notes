## 6. ANONYMITY TOOLS

---

## proxychains

Routes traffic through proxy servers.

### Used for:

- hiding real IP address  
- chaining multiple proxies  
- routing tools like Nmap, Hydra through anonymity layers  

---

## tor

Anonymous network routing system.

### Used for:

- anonymous browsing  
- hiding user location  
- accessing .onion services  
- traffic encryption through multiple relays  

---

## macchanger

MAC address spoofing tool.

### Used for:

- changing network interface identity  
- bypassing MAC-based filters  
- anonymizing device on local networks
